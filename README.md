## Description

TaskTreasure is a web application designed to help you manage your tasks and stay organized. With TaskTreasure, you can:

- Add new tasks
- Mark tasks as completed

## Run Locally

```shell
git clone https://github.com/your-username/taskTreasure.git
cd taskTreasure
npm install
npm run dev
```
